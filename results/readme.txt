ASTER generates 11 columns results:
Column 5th: the state of second column gene, either amplification or deletion
Column 6th: the state of fourth column gene, either amplification or deletion
Column 7th, 8th, and 9th: 1st pvalue, 2nd pvalue and 3rd pvalue of the test results.
Column 10th: number of samples belong to the column 5th state (amplification or deletion)
Column 11th: number of samples belong to the column 6th state (amplification or deletion)
